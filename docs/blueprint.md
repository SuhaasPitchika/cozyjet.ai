# **App Name**: CozyJet.AI

## Core Features:

- AI Marketing Content Generation (Snooks): Users interact with the Snooks agent via a chat interface to generate high-quality, multi-platform marketing content (e.g., LinkedIn posts, X tweets, emails, YouTube scripts), leveraging advanced AI strategies and personalized contexts.
- AI Productivity Timeline & Scoring (Flippo): Flippo analyzes client-side summarized activity data to generate a structured work timeline and a deep work score, visualizing periods of focus versus distraction, providing insights into productivity patterns.
- Local AI Screen Intelligence (Skippy): Skippy provides privacy-centric, real-time screen intelligence by analyzing user activity locally (via ONNX vision model and Tesseract OCR) and uses a client-side LangChain tool to offer contextual assistance or prompts when the user is stuck.
- Interactive Agent Dashboard: A dynamic dashboard provides a central hub for users to activate, monitor, and interact with Skippy, Flippo, and Snooks, including their visual representations and specific controls.
- Secure Client-Side Data Storage: All sensitive screen analysis data generated by Skippy is processed and stored exclusively client-side in an encrypted IndexedDB, ensuring maximum user privacy and control over their data.
- Comprehensive User Authentication & Security: Robust user authentication (Google OAuth, custom credentials) and authorization, ensuring secure access with features like JWTs, refresh token rotation, and HTTP-only cookies, safeguarding user accounts.
- Animated Cinematic Landing Page: An engaging, visually rich landing page built with modern animation libraries (GSAP, Framer Motion, Three.js) to showcase the unique capabilities and 'living' nature of CozyJet.AI and its agents.

## Style Guidelines:

- The overall aesthetic emphasizes a sophisticated, tech-forward feel with depth. Primary elements will use a vibrant purple: '#A36BEE' (RGB: 163, 107, 238) to convey innovation and a touch of creativity. This color is prominent against darker backgrounds and offers versatility for a range of UI elements.
- Backgrounds predominantly utilize a deep, subtle purple-gray: '#231F2A' (RGB: 35, 31, 42). This dark hue provides a premium, 'cozy' yet modern base, allowing glassmorphic elements and animated effects to stand out with clarity and contrast, while complementing sections with white backgrounds.
- Accent components, interactive states, and highlights will feature a bright, energetic blue: '#6297FF' (RGB: 98, 151, 255). This provides a striking visual counterpoint to the primary purple and deep background, drawing attention to calls-to-action and key functionalities, aligning with the 'jet' aspect of the brand.
- Headlines and prominent text use 'Space Grotesk' (sans-serif) for its modern, tech-inspired, and slightly futuristic feel, resonating with the AI and innovation theme. Body text and longer content blocks will employ 'Inter' (sans-serif), ensuring excellent readability and a clean, objective appearance.
- For any displays of computer code, 'Source Code Pro' (monospace sans-serif) will be used to maintain a clear, consistent, and developer-friendly presentation.
- Icons will incorporate a blend of playful and sophisticated styles, including clean 3D-style SVGs for platform integrations (e.g., YouTube, X), custom animated elements like Skippy's googly eyes, and subtle visual cues like small drop shadows for realism.
- The layout is characterized by modern glassmorphism, full-width sections, and adaptable components. Key elements include a fixed, translucent navigation bar, a multi-column dashboard with a fixed sidebar, sticky-note style info cards, and distinct card layouts for agents and content outputs, emphasizing visual hierarchy and user focus.
- Extensive use of cinematic animations defines the user experience, powered by Framer Motion, GSAP, and Three.js. This includes viewport-triggered transitions, staggered text animations, 3D element interactions (parabolic image arcs, tilt cards), particle effects, real-time drawing lines, infinite scroll tickers, SVG morphing, and subtle character micro-animations, ensuring a 'living' and engaging interface with a target of 60fps.
